Hidden Settings

Show iMessage Debug Settings

Open /var/mobile/library/preferences/.GlobalPreferences.plist

Add a new value called "iMessageDiagnostisEnabled" set it to boolean and yes

Now restart or respring and your done

Show hidden safari sharesheet options                                     

Open /var/mobile/library/preferences/.GlobalPreferences.plist

Add a new value called "SafariDebugOverrideLaunchMode" set its value to number and set the number to zero

Now restart or respring and your done

Show RSSI in status bar

Open /var/mobile/library/preferences/.GlobalPreferences.plist

Add a new value called "SBShowRSSI" set its value to boolean and set the value to yes

Now restart or respring and your done

Show build number in status bar

Open /var/mobile/library/preferences/.GlobalPreferences.plist

Add a new value called "UIStatusBarShowBuildVersion" set it to boolean and yes

Now restart or respring and your done

Show itens tab in find my permanently                              

Open /var/mobile/library/preferences/com.apple.findmy.fmipcore.notbackedup.plist

find the value "savedItemsTabEnabled" if it is not there add it, then set the boolean value to yes

find the value "itemsTabEnabled" if it is not there add it, then set the boolean value to yes

Now restart or respring and your done

Show stats in mesure app

Open /var/mobile/library/preferences/com.apple.measure.plist

Find the value "kShowStatistics" and set its boolean value to yes

Now restart or respring and your done
